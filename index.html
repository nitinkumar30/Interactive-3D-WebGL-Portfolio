The Gallery: Selected Automation & Development Projects

In the 3D gallery room, you can walk through an exhibition of my most significant automation and development projects. Here is a detailed breakdown of the work created by Nitin S Kumar.

Auto Edu Mail Generator

Auto Edu Mail Generator is an expansive Python automation tool that streamlines the generation of educational institute emails. With 119 stars on GitHub, this is my highest-starred project. Built with Python and Selenium, it automates the bulk creation of educational email accounts, saving hours of manual effort. The project features a modular architecture, robust error handling, and support for multiple educational domains.

Naukri Auto Resume Uploader

Naukri Auto Resume Uploader solves a real-world problem — automating resume uploads on the Naukri job portal. Built with Python and Selenium, this tool automates the tedious process of regularly updating your resume on job portals. It features headless browser automation, scheduled execution, and intelligent form handling. This project demonstrates my ability to identify real-world friction points and eliminate them through smart automation.

n8n Workflow Automation Projects

The n8n Projects collection showcases my expertise in workflow automation using the n8n platform. It includes a series of automated workflows connecting various services and APIs to create powerful automation pipelines. This project demonstrates proficiency in TypeScript, API integration, and modern workflow orchestration — essential skills for any automation engineer.
